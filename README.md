# Mediant UI Shell

This repository is a polished, front-end-only visual shell for `Mediant`.

It intentionally does not include:

- backend logic
- subscriptions
- AI features
- real uploads
- authentication

It is just a clean product-direction mock so the app can be seen and discussed.

## Run locally

Open `index.html` in a browser, or serve the folder with a simple static server such as:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.
